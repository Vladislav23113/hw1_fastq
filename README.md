Парсер работает так: в конструкторе вызывается метод `computeReadingStatistic()`, который за один проход по файлу заполняет все основные поля-характеристики прочтений.
Реализация метода `trim(seq, qual, w, q)` была полностью взята у Trimmomatic. Весь остальной код, где необходимо, снабжен комментариями.
